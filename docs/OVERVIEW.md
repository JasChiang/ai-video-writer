# 專案功能總覽

AI Video Writer 專注於「以 YouTube 影片為核心的內容再利用」，讓影片內容可被快速轉換成文章、分析報告與成長策略。

## 主要使用情境

- 影片內容轉長文：從影片語意生成文章與配圖，提升 SEO 與內容覆蓋。
- 影片中繼資料優化：大量生成標題、描述、關鍵字與發布建議。
- 頻道經營分析：透過 YouTube Analytics 與 AI 分析找出成長機會。
- 關鍵字/主題分析：產生題材策略與內容結構建議。

## 功能模組

### 1. 文章生成
- 來源：YouTube 影片或 URL。
- 輸出：文章、截圖、段落重點與配圖位置建議。
- 支援：自訂模板、參考資料與檔案上下文。

### 2. 影片中繼資料生成
- 影片標題、描述、關鍵字建議。
- 批次與單片模式。

### 3. 頻道與影片分析
- 頻道儀表板（整體指標、趨勢、觀眾與來源）。
- 影片表現分析（強勢/弱勢樣本、內容成效診斷）。
- 影片長度分析（長度與續看/成效關聯）。

### 4. AI 分析與建議
- 多模型 AI：Gemini / OpenRouter。
- 串流分析：分析進度與階段式回饋。
- 多種分析類型：綜合分析、內容策略、觀眾洞察、關鍵字分析等。

### 5. 快取與配額優化
- 影片快取透過 GitHub Gist 儲存。
- 分析與搜尋優先使用快取，節省 YouTube API 配額。

### 6. Notion 發佈
- 生成內容可直接發佈至 Notion 資料庫。
- 支援 OAuth 與 Integration Token 模式。

## 你可以期待的輸出

- SEO 文章（含截圖）
- 影片標題/描述/關鍵字
- 頻道經營報告（策略、優勢、弱點、行動建議）
- 影片分析建議（低效影片診斷、優化方向）
